课程简介：

KVM 是Kernel-based Virtual Machine的简称，是一个开源的系统虚拟化模块。KVM的虚拟化需要硬件支持（如intel vt技术或者AMD V技术)。是基于硬件的完全虚拟化。虚拟化可以虚拟出来多个操作系统：每个操作系统之间是相互独立的。

本章详细讲解KVM企业级应用部署管理。学习完本次课程学员可以独立完成企业kvm部署 克隆  快照 文件系统格式相互转换
